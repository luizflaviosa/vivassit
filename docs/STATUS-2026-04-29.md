3. **VAPID keys não setadas** — push notifications infraestrutura existe mas não envia.

### Altas (devem entrar Onda 2.5 final)

5. **`/api/auth/verify-otp` endpoint** existe mas não vi código nem teste. Login OTP depende dele funcionar.

8. **OTP expiry padrão 3600s** no Supabase Auth (recomendado 600s).
9. **Postgres versão antiga** (vulnerable per advisor) — upgrade pendente.

### Baixas (cosmético / refactor)
