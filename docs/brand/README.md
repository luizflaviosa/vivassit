# Singulare — Brand Books

Dois sistemas visuais coexistem na marca Singulare. Um é oficial e cobre 90%
dos contextos. O outro é paralelo, para comunicação 1:1 e peças editoriais.

## Arquivos

### `system-luxury-modular-v1.html` — sistema oficial
**Status:** ainda não versionado no repo. Ver `luxury-modular-v1-PLACEHOLDER.md`
para instruções de como adicionar.

- **DNA:** Luxury Modular Intelligence
- **Paleta:** navy escuro `#0F1B33` + gold `#FFC62F` + sand
- **Logo:** 3 quadrados sobrepostos
- **Tipografia:** Poppins Bold
- **Use em:**
  - Produto digital — app, painel admin, landing pages
  - Comunicação corporativa, B2B
  - Materiais para Apple Developer Program
  - Sales deck, pitch, mídia paga

### `system-editorial-warm-v1.html` — sistema paralelo
**Status:** documentado neste diretório.

- **DNA:** Editorial Warm — premium calmo, terra + água
- **Paleta:** navy `#1F3D4D` + terracota `#B85C3A` + cream `#ECE2CB` + off-white `#F8F4EB` + slate `#6B7077` + graphite `#1F1F1F`
- **Tipografia:** Space Grotesk Medium (display) + Poppins Regular (body)
- **Use em:**
  - Comunicação 1:1 — email welcome, ofertas, agradecimentos
  - Peças editoriais long-form, manifesto
  - Materiais impressos — papelaria, dossiê, postal
  - Variações de campanha warm (sazonal, retrospectiva)

### `email-welcome-editorial-warm.html` — preservação
Cópia do email welcome v1 alinhado ao sistema Editorial Warm. **Arquivo de
referência apenas** — não é template ativo. O template em produção no n8n
agora é Luxury Modular.

## Regra de decisão

Pergunta única: **Esta peça é arquitetura ou carta?**

- **Arquitetura** (produto, sistema, B2B, corporativo) → Luxury Modular v1.0
- **Carta** (1:1, editorial, impresso, warm) → Editorial Warm v1.0

Na dúvida, defaulta pro Luxury Modular — ele é o oficial.

## Não fazer

- Não misturar paletas entre sistemas na mesma peça.
- Não usar Editorial Warm em produto digital (app, painel) — fica fora de tom.
- Não usar Luxury Modular em email 1:1 — fica frio demais.
- Não criar terceiro sistema sem decisão explícita do owner.
